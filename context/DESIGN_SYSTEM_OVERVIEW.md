# Design System Overview — Nooc

## Brand vibe
Calm, premium, minimal, cozy

## Goals
- Availability and price visible immediately
- Quiet, distraction-free UI
- High trust, low friction

## Layout
- Mobile-first
- Bottom navigation
- Explore defaults to list + map toggle

## Venue cards
- Name
- Neighborhood
- $ / seat / hour
- Availability status
- 2–4 trust tags

## Visual style
- Dark green primary color
- Warm neutrals
- Soft borders and rounded corners
- Large readable typography

## Typography
- Clean modern sans-serif
- Editorial spacing

## Components
- VenueCard
- Tag/Pill
- AvailabilityPicker
- SeatStepper
- PrimaryButton

## Voice
- Calm
- Confident
- Professional
