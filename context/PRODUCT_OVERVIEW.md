## Core thesis
Nook is Resy for workspaces: reserve a seat by the hour in calm, professional public environments.

## Inventory model
- Venue
- Tables within venue
- Seats per table
- Reservations block seat inventory by time

## Pricing
- Hourly price per seat
- Venue-defined rules (stored as text/JSON for future automation)

## User flow
1. Browse venues
2. View availability and price
3. Select time, duration, seats
4. Instant confirm
5. View reservation

## Venue flow
1. Create venue profile
2. Define hours
3. Create tables and seats
4. Set pricing
5. Publish

## Trust signals
- Quiet suitability
- Wi-Fi
- Outlets
- Call policy

## MVP exclusions
- Payments
- Reviews
- Messaging
- Loyalty
- Advanced maps

## Roadmap scaffold
- v1: List + map + booking
- v1.5: Filters and richer trust signals
- v2: Payments and enforcement
- v3: Reviews and subscriptions
